# Renewable Energy, Economic Growth, and CO₂ Emissions Analysis

## Overview
This project investigates the relationship between renewable energy consumption (REC), economic growth (GDP), and carbon dioxide (CO₂) emissions across nine developed and developing countries from 2015 to 2024. The analysis follows a structured data analytics workflow that includes data cleaning, exploratory data analysis (EDA), statistical analysis, regression modeling, and data visualization.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the required packages using the following command:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
- Open the notebook in the notebooks folder to run the full analysis workflow.
- Use the Python scripts in the src folder for data loading, preprocessing, analysis, and visualization.
- Processed outputs can be saved in the data/processed folder after running the analysis.

## Key Findings
The analysis explores how REC and GDP relate to CO₂ emissions across a panel of countries, with a focus on comparing developed and developing economies and highlighting policy-relevant insights.
